Question #1

Topic: Machine Learning Fundamentals

What is the primary purpose of cross-validation in machine learning?

A. To increase the size of the training dataset
B. To evaluate model performance and reduce overfitting
C. To speed up the training process
D. To automatically select features

Correct Answer: B

Explanation: Cross-validation is used to assess how well a model generalizes to unseen data and helps detect overfitting by testing the model on multiple train/validation splits.

Which Google Cloud service is best suited for training large-scale machine learning models?

A. Cloud Functions

Question #3

Topic: Model Deployment

What is the main advantage of using A/B testing for model deployment?

A. It reduces training time
B. It allows gradual rollout and performance comparison
C. It automatically fixes model bugs
D. It increases model accuracy

Correct Answer: B

Explanation: A/B testing allows you to deploy a new model to a subset of users while keeping the existing model for others, enabling performance comparison and risk mitigation.
